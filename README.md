# KernelHWOtus
Домашнее задание по теме "Сборка ядра"
В задании требовалось выполнить:
1. Запустите ВМ c Ubuntu.
2. Обновите ядро ОС на новейшую стабильную версию из mainline-репозитория.
3. Оформите отчет в README-файле в GitHub-репозитории.
В первую очередь ставим ОС: Я решил использовать ОС Ubuntu 24.04 LTS на ВМ VirtualBox. Выделяю я ей 4 ГБ ОЗУ и 4 ядра, 25 ГБ памяти диска, подключаю сетевой мост на мой LAN адаптер.
Установив ОС можно приступать к работе.
В первую очередь мы узнаём версию ядра:
uname -r
У меня результат:
6.8.0-101-qeneric
Создаём папку и заходим в неё (для удобства назвал её kernel):
mkdir kernel && cd kernel
Заходим на браузер репозиториев и ищем интересующую нас версию ядра. Я выбрал 6.16.2. Проваливаемся внутрь и ищем акутальные ссылки для скачки пакетов. Важно учесть нашу архитекутуру процессора. В моём случае я скачачиваю amd64. С помощью wget скачиваем ссылки:
wget https://kernel.ubuntu.com/mainline/v6.16.2/amd64/linux-headers-6.16.2-061602-generic_6.16.2-061602.202508201745_amd64.deb
wget https://kernel.ubuntu.com/mainline/v6.16.2/amd64/linux-headers-6.16.2-061602_6.16.2-061602.202508201745_all.deb
wget https://kernel.ubuntu.com/mainline/v6.16.2/amd64/linux-image-unsigned-6.16.2-061602-generic_6.16.2-061602.202508201745_amd64.deb
wget https://kernel.ubuntu.com/mainline/v6.16.2/amd64/linux-modules-6.16.2-061602-generic_6.16.2-061602.202508201745_amd64.deb
После чего устанваоиваем пакеты:
sudo dpkg -i *.deb 
И проверяем наше ядро:
ls -al /boot
Теперь можно перезапустить и выбрать нужное, после чего мы проверяем какую версию ядра мы используем теперь:
uname -r
Сейчас у меня результат:
6.16.2-061602-generic
Все мои шаги, выполненные в  данном задании показаны на скриншотах в той же директории, что и этот README.
